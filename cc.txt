🧪 EXPERIMENT 7: AWS Lambda + S3 → DynamoDB
🎯 Goal:

When a file is uploaded → Lambda runs → data stored in DynamoDB

🪜 Steps:
Login to AWS Console
Open Lambda
Click Create Function
Choose:
Author from scratch
Runtime → Python
Create IAM role (basic permissions)
Paste the given Python code
Add Trigger:
Select S3
Choose bucket
Event → “Object Created”
Click Deploy
Upload file into S3 bucket
Check DynamoDB → data inserted

🧪 EXPERIMENT 9: Load Balancer + Auto Scaling
🎯 Goal:

Distribute traffic across servers

Part A: Load Balancer
🪜 Steps:
Launch 2 EC2 instances

Install Apache:

yum install httpd -y
systemctl start httpd
Add HTML:
Server 1 → “This is Server 1”
Server 2 → “This is Server 2”
Create Load Balancer
Choose:
Application Load Balancer
Internet facing
Create Target Group
Register both EC2 instances
Copy DNS → open in browser
Refresh → servers switch

👉 Load balancing works ✅

Part B: Auto Scaling
🪜 Steps:
Create AMI from EC2
Create Launch Template
Create Auto Scaling Group
Set:
Min = 1
Max = 4
Desired = 2
Attach Load Balancer
Test:
Terminate instance
New instance auto-created

👉 Auto scaling works ✅
