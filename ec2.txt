1. Launch an EC2 Instance (Ubuntu)

Steps:

Go to AWS Management Console → EC2.
Click Launch Instance.
Enter instance name → “MyUbuntuAppServer”.
Choose Ubuntu Server 22.04 LTS (Free Tier).
Choose Instance type → t2.micro (Free Tier eligible).
Create a Key Pair → mykey.pem (download it).
Configure Security Group:
Allow SSH (port 22) — your IP.
Allow HTTP (port 80) — anywhere.
If app runs on another port (e.g., 3000, 8080), allow that too.
Launch the instance.


2. Install Docker and Required Software

SSH into EC2
  ssh -i "mykey.pem" ubuntu@54.123.98.11
Update packages
  sudo apt update && sudo apt upgrade -y
Install Docker
  sudo apt install docker.io -y
Install Git
  sudo apt install git
Install nano
  sudo apt install nano

3. Clone Your GitHub Application
git clone https://github.com/budarajumadhurika/AWSExample.git

4. Write the Dockerfile
nano Dockerfile
  FROM nginx:alpine

5. Build the Docker Image & Run It
docker build -t myapp-image .
docker run -d -p 3000:3000 myapp-image
